# Родовая навигация

Статический сайт без сервера и без ИИ. Все расчёты выполняются локально в браузере.

## Что положить в GitHub / Vercel

В корне репозитория должны лежать два файла:

```text
index.html
favicon.svg
```

После замены этих файлов в ветке `main` Vercel автоматически опубликует новую версию.

## Что считает текущая версия

- числовое ядро, семейство и архетип;
- числовые связи с матерью, отцом и между родителями;
- режим родительского присутствия, который задаёт пользователь;
- годовой столп Ба-цзы: животное, элемент, отношения У-син, союзы и оппозиции животных;
- повторения чисел, животных и элементов до трёх поколений;
- упрощённый слой Ло Кхак на следующие девять лет;
- синтез совпадающих сигналов из разных слоёв.

## Важное ограничение

В восточном слое сейчас используется годовой столп, а не полный классический Ба-цзы с месяцем, днём и часом. Для дат 1–4 февраля сайт отмечает необходимость вручную сверить китайский Новый год.
